Szül.idő:2006.11.13

Aluláteresztő RC szűrő

*Adatok:*

$R = 11 Ohm $   
$C = 13 uF $

*Határfrekvencia:*

$fh= {1 \over (2 * Pi * R * C)} $   
$fh= {1 \over (2 * Pi * 11R * 13uF)} $   
$fh= 1112,9761 Hz $
